## Frequently Asked Questions

**What makes the on-page agent different from a regular chatbot?**
Most chatbots sit on top of a static knowledge base. ShopGuide's on-page agent connects live to the Shopify Catalog API. It sees real variants, metafields, inventory, and pricing in real time. That is the difference between guessing and knowing.

**Does it work with large catalogs?**
Yes. The agent is built for high-SKU stores. It indexes the full product graph so long-tail items surface through natural language instead of exact keyword matches.

**How does it help Average Order Value?**
By removing discovery friction and making confident recommendations, customers build larger, more complete baskets. Live stores have seen assisted orders with significantly higher AOV than the store-wide average.

**Is setup complicated?**
No. Install from the Shopify App Store, add the theme block, and the agent starts learning from your live catalog. Custom CSS keeps it on-brand.

**Will it replace my support team?**
No. It handles the repetitive discovery and product questions that eat support time. Complex or edge-case issues still go to humans. That is the honest trade-off.
