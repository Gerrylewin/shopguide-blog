## From Passive Browsing to Measurable Assisted Revenue

Every merchant with a sizable catalog knows the frustration. A customer lands on your site, spends time clicking around, adds a few items… and then disappears. They were *interested*, but the path from interest to purchase was unclear.

Traditional tools put the burden on the shopper. Path Explorer puts the intelligence in the agent.

**Path Explorer** is the feature inside ShopGuide that turns raw browsing signals and chat intent into a dynamic, guided path to purchase. It doesn't just recommend — it classifies where the shopper is in their journey and chooses the highest-confidence next action.

## What Path Explorer Actually Does

Path Explorer continuously observes on-page actions (views, filters, dwell time, comparisons) + explicit chat inputs. It then:

- Detects and tags shopper intent
- Classifies the current funnel stage (discovery, comparison, validation, checkout)
- Selects and executes the best next action (clarifying question, focused shortlist, objection handling, add-to-cart nudge, checkout push)
- Updates the shopper profile and path in real time as new signals arrive
- Logs every action and outcome for measurement

The result is a living, intent-driven conversation that reduces mid-funnel friction and creates trackable assisted conversions.

## Core Components

**1. Signal Capture & Intent Parsing**  
Pulls from Shopify storefront events + chat. Outputs intent tags + confidence scores.

**2. Stage Classification**  
Uses behavior timeline + intent to score where the shopper is (Discovery → Comparison → Validation → Checkout).

**3. Action Selector & Response Generator**  
Chooses the optimal next step and generates the agent response or store action (recommendation, cart add, policy answer, etc.).

**4. Outcome Tracking & Analytics**  
Every action is logged. Outcomes (add-to-cart, purchase, bounce) flow into dashboards for assisted revenue, CVR lift, AOV delta, time-to-cart, and objection resolution rates.

**5. Guardrails & Experimentation**  
Confidence thresholds gate high-impact actions. Falls back to safe guidance when uncertain. Designed for A/B testing and iterative tuning.

## Real Integration Points

Path Explorer connects directly to:
- Catalog + Inventory API (accurate, in-stock recommendations)
- Cart/Checkout API (seamless add-to-cart and nudge)
- Reviews API (social proof at the right moment)
- Lifecycle/Email API (follow-up triggered by assisted actions)
- Analytics + Experimentation layer (KPI dashboards and lift measurement)

## Frequently Asked Questions

**How is Path Explorer different from simple product recommendations?**  
Traditional recs are often static or history-based. Path Explorer is live, stage-aware, and action-oriented — it chooses *what to do next* based on where the shopper is in their journey.

**Does it require custom development?**  
No. It works with standard Shopify APIs and the ShopGuide agent. Merchants instrument events once; the rest is handled inside the platform.

**How do I measure the impact?**  
Dashboards show assisted CVR, revenue per assisted session, AOV delta, time-to-cart, and objection resolution rates. A/B tests are built in.

**What happens when confidence is low?**  
Safe fallback to general guidance or clarifying questions. High-impact actions (auto add-to-cart) are gated behind confidence thresholds.

**Can it handle long-tail and niche products?**  
Yes — by matching specific intent signals to the exact products and complementary items that complete the buyer's project.
